# LM2596-12V-to-5V-Power-Supply-PCB
A custom 12V to 5V switching power supply PCB using LM2596, designed and validated in KiCad.

A custom DC-DC buck converter PCB designed using KiCad. This project converts a 12V DC input into a stable 5V output using the LM2596 switching voltage regulator.

## Features

- 12V DC Input
- Stable 5V Output
- LM2596 Switching Regulator
- Input and Output Connectors
- Through-Hole Components
- 2-Layer PCB Design
- ERC and DRC Validated
- Manufacturing Ready Gerber Files

## Applications

- ESP32 Development Boards
- Arduino Projects
- Sensor Modules
- Embedded Systems
- General Electronics Projects

## Software Used

- KiCad

## Design Flow

1. Created schematic in KiCad
2. Assigned footprints
3. Performed ERC validation
4. Designed PCB layout
5. Routed traces and added ground plane
6. Performed DRC validation
7. Generated Gerber files
